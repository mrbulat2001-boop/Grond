# Земли Бри — прототип игровой карты

Первая часть продолжения `Shire 2D RPG v7.1`, собранная как отдельная сцена для Godot 4.2.

## Уже есть

- нарисованная SVG-карта 2400×1600 в стилистике исходного проекта;
- Бри, Стэддл, Комб, Арчет, Четвуд, топи Миджуотер и западная кромка Могильников;
- Великий Восточный тракт и Зелёный тракт;
- перемещение, камера, определение области, интерактивные ориентиры;
- атлас по `M`, ожидание по `T`, дневная и ночная палитра;
- сюжетные зацепки вокруг исчезнувших обозов и тревожных огней в Четвуде.

## Запуск

Откройте `project.godot` в Godot 4.2 и запустите проект.

## Управление

`WASD` / стрелки — движение, `E` — взаимодействие, `M` — атлас, `T` — подождать час, `Esc` — закрыть окно.

Следующий этап: перенести из исходного проекта менеджеры квестов, сохранений, живых NPC, фонарей и дорожных событий, не меняя утверждённую географию.
